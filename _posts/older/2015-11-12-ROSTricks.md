# Templates

## Launch file

Example for a file named "some_skill.launch":

{% highlight xml %}
<?xml version="1.0" encoding="UTF-8"?>

<launch>
    <arg name="robot" />

    <group ns="$(arg robot)">
        <node name="some_skill" pkg="some_skill" type="node.py" output="screen" />
    </group>
</launch>
{% endhighlight %}
